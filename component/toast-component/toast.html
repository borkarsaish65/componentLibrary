Toasts/Snackbars

Toasts are popups used to show message on the screen.

Default Toasts

Default toasts will appear on the top right section of the screen.

The toast will last for 2 seconds and disapear.

Code Example 

         <div  id="taost-1"  class="toast-container hide">
         <div class="toast-header">
             Smart UI Toast
         </div>
         <div class="toast-body">
             <p> This is a basic toast!</p>
         </div>
                    </div>

The hide class in the toast need to be toggled with show class in order to show the toast.

Toast in the center

Toast can be made to appear at the center of the screen by using 'center' class.

Code Example 

 <div  id="taost-1"  class="toast-container center hide">
 <div class="toast-header">
     Smart UI Toast
 </div>
 <div class="toast-body">
     <p> This is a basic toast!</p>
 </div>
            </div>
Javascript Code Example

If the toast is attached on the button onclick event need to be bind with openToast function. The button also need to have data-target attribute set to the id of the taost which needs to be displayed.


                function openToast(event){

                    let target = event.target;
                    let id = target.getAttribute('data-target');
                    let toast = document.querySelector(id);
                
                    toast.classList.remove('hide')
                    toast.classList.add('show')
                
                    setTimeout(()=>{
                        toast.classList.remove('show')
                        toast.classList.add('hide')
                
                    },2000)
                }
