Installation Guide

CSS

The UI library can be easily installed by placing the below code in the head tag.


    <link rel="stylesheet" href="https://pro.fontawesome.com/releases/v5.10.0/css/all.css" integrity="sha384-AYmEC3Yw5cVb3ZcuHtOA93w35dYTsvhLPVnYs9eStHfGJvOvKxVfELGroGkvsg+p" crossorigin="anonymous"/>
    <link rel="stylesheet" href="../smartui.css">
JS

Some of the components require the use of JavaScript to function. The below script can be placed just before the body tag ends. Make sure to place the script before your other JS scripts.


                        <script src="smartui.js"></script>
Starter Template

Below is the starter template code which can be used to build your app on top of it. Happy Coding!


                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta http-equiv="X-UA-Compatible" content="IE=edge">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <link rel="stylesheet" href="https://pro.fontawesome.com/releases/v5.10.0/css/all.css" integrity="sha384-AYmEC3Yw5cVb3ZcuHtOA93w35dYTsvhLPVnYs9eStHfGJvOvKxVfELGroGkvsg+p" crossorigin="anonymous"/>
                            <link rel="stylesheet" href="../smartui.css">
                            <title>Smart UI</title>
                        </head>
                        <body>
                        
                            <h1 class="h1">Hello World</h1>
                            
                            <script src="smartui.js"></script>
                        </body>
                        </html>
