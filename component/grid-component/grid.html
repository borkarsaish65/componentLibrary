Grid

Grid is a 2D grid based layout system which makes use of rows and columns to make designing web pages easy.

Grid with two items.

This is a basic grid layout containing two child elements.

This layout can be created with grid-column-2 and grid-container class together.

Code Example 

                    <div class="grid-example">
                    <div class="grid-container grid-column-2 grid-gap-example">
                        <div class="grid-items grid-item-example">
                        </div>
                        <div class="grid-items grid-item-example">
                        </div>
                    </div>
                </div>
