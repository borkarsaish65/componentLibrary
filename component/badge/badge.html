Badges

Badges are used to notify status of the user or it can also be used to show notification number.

Avatar with badges

Badge can be used with smart ui avatar component easily. Below is a sample code to demonstrate that.

Code Example 

      <div class="badge-container badge-relative">
      <img class='avatar' src="https://picsum.photos/200/300" alt="pic">
       <span class="badge bg-online badge-avatar-default"></span></div>
Buttons with badges

Badges can also be used along with buttons. Below code demonstrating the same.

Below code demonstrating button with badge placed in top right corner.

Code Example 

            <button class="btn btn-primary badge-relative">
            Inbox <span class="badge bg-danger bg-top-right">5+</span>
            </button>


            <button class="btn btn-primary badge-relative">
            Inbox <span class="badge bg-danger bg-top-right"> <i class="far fa-bell"></i> </span>
            </button>

Below code demonstrating button with badge placed next to the button text.
