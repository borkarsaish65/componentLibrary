Avatar

Avatar component used to display profile picture of user.

Default Avatar
Code Example

                    <img class='avatar' src="./avatar.jpg" alt="pic">
Circular Avatar
Code Example

                    <img class='avatar circle' src="./avatar.jpg" alt="pic">
Extra Small Avatar

You can control the size of avatar using other avatar classes.

Below is the extra small avatar which can be achieved using the class 'xxsmall' along with the default avatar class.

Code Example
<img class='avatar xxsmall' src="./avatar.jpg" alt="pic">
Small Avatar

Below is small avatar, which can be achieved using "small" class.

Code Example
<img class='avatar small' src="./avatar.jpg" alt="pic">
Large Avatar

Below is large avatar, which can be achieved using "large" class.

Code Example
<img class='avatar large' src="./avatar.jpg" alt="pic">
Extra large Avatar

Below is Extra large avatar, which can be achieved using "xxlarge" class.

Code Example
<img class='avatar xxlarge' src="./avatar.jpg" alt="pic">
Combination Avatar

You can combine above avatar classes to get desired avatar.

Below is an example of using circle and xxlarge classes together on a img tag.

Code Example
<img class='avatar xxlarge circle' src="./avatar.jpg" alt="pic">
