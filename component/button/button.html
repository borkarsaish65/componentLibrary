Buttons

Smart UI library provides classes which can be used to enhance the styles of buttons.

Examples
Primary Button
Secondary Button
Success Button
Danger Button
Link Button
Code Example 

            
Primary Button

       <button type='button' class='btn btn-primary'>Primary</button>

            
Secondary Button

       <button type='button' class='btn btn-secondary'>Secondary</button>

            
Success Button

       <button type='button' class='btn btn-success'>Success</button>

            
Danger Button

       <button type='button' class='btn btn-danger'>Danger</button>

            
Link Button

       <button type='button' class='btn btn-link'>Click</button>
Outline Buttons

Outline buttons removes all background color and gives it a lighter look.

Primary Button
Secondary Button
Success Button
Danger Button
Code Example 

            
Primary Button

    <button type='button' class='btn btn-outline-primary'>Primary</button>

            
Secondary Button

    <button type='button' class='btn btn-outline-secondary'>Secondary</button>

            
Success Button

    <button type='button' class='btn btn-outline-success'>Success</button>

            
Danger Button

    <button type='button' class='btn btn-outline-danger'>Danger</button>
Icon Buttons

Icon button helps in giving extra information about the button by having icon present in the button.

Icon Button

Code Example 

     <button type='button' class='btn btn-primary'>
         <i class="far fa-address-card"></i>   
      Generate ID
    </button>
Floating Action Button

Floating action button acts as the call to action button on the screen.

It performs the primary action on a screen.

Floating Action Button

Code Example 

   <button type='button' class='btn-floating'>
       <i class="far fa-bell"></i>
   </button>
