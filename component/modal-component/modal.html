Javascript Code Example

Below is the javascript code used to display the modal. The modal-wrapper can be toggled with 'show' class to view and hide the modal.


                    function openModal(event){
                        let target = event.target;
                        let id = target.getAttribute('data-target');
                        let modal = document.querySelector(id);
                        modal.classList.add('show');
                    }  
                    function closeModal(e){
                        let parentElement = e.target.parentElement.parentElement.parentElement;
                        parentElement.classList.remove('show');
                    }
