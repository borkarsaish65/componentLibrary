Default Input Field

This field will take up the entire available size horizontally.

Code Example 

    <div class="formDiv">
    <label for="Name">Enter Name</label>
    <input class="input" type="text">
    </div>
Input Field with only bottom border

In this type of input field only the bottom border will be visible

Code Example 

          <div class="formDiv">
          <label for="Name">Enter Name</label>
          <input class="input bottom-border" type="text">
          </div>
Input Field with Error

Error in the input can be notified to user using the class "error".

Code Example 

          <div class="formDiv">
          <label for="Name">Enter Name</label>
          <input class="input error" type="text">
          </div>
Input field with bottom border with Error

"error" class is also compatible bottom border input fields.

Code Example 

          <div class="formDiv">
          <label for="Name">Enter Name</label>
          <input class="input bottom-border error" type="text">
          </div>
Sample Form with error tooltip

Tooltip can be used to display the error message for the user input.

Please enter name.
Please enter age.
Code Example 

                <div class="formDiv">
                <div class="flex">
                    <div class="section position-relative">
                        <label for="Name">First Name</label>
            <input class="input bottom-border error" type="text">
                        <div class="invalid-msg">
                            Please enter name.
                        </div>
                    </div>
                    <div class="section position-relative">
                        <label for="Name">Age</label>
            <input class="input bottom-border error" type="text">
                        <div class="invalid-msg">
                            Please enter age.
                        </div>
                    </div>
                </div>
                </div>
