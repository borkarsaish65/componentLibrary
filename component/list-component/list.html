List

List component allows us to show list of information together in an asthetic way.

Spaced List

Below is the basic implementation of spaced list.

    Spaced List
  • Item A
  • Item B
  • Item C
  • Item D
Code Example

Below is the basic implementation html code of list.


                    <div class="list-collection-container">
    
                        <ul class="list-collection list-style-none">
                            <div class="list-heading">
                               Spaced List
                            </div>
                            <li class="list-item">Item A</li>
                            <li class="list-item">Item B</li>
                            <li class="list-item">Item C</li>
                            <li class="list-item">Item D</li>
                        </ul>
                    </div>
Stacked List

Stacked list are list with spaces between each list item.

    Stacked List
  • Item A
  • Item B
  • Item C
  • Item D
Code Example

Below is the basic implementation html code of Stacked list.


                    <div class="list-collection-container">
    
                    <ul class="list-collection list-style-none list-stacked">
                        <div class="list-heading">
                           Stacked List
                        </div>
                        <li class="list-item">Item A</li>
                        <li class="list-item">Item B</li>
                        <li class="list-item">Item C</li>
                        <li class="list-item">Item D</li>
                    </ul>
                
                </div>
Active List Item

You can set active state on the stack element. You can use 'active' class to show the active state of list item.

    Stacked List
  • Item A
  • Item B
  • Item C
  • Item D
Code Example 

                    <div class="list-collection-container">
    
                    <ul class="list-collection list-style-none list-stacked">
                        <div class="list-heading">
                           Stacked List
                        </div>
                        <li class="list-item">Item A</li>
                        <li class="list-item">Item B</li>
                        <li class="list-item active">Item C</li>
                        <li class="list-item">Item D</li>
                    </ul>
                
                </div>
